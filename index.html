Spotify Web App

My Spotify web app allows you to view your most played songs. To do this, the app requires you to authenticate through Spotify. After authentication is complete, you can view your most played songs from the last four weeks to the last several years. You can also select the number of songs to show, ranging from 10 to 50. This information is gathered through Spotify's API. The app itself is built with React.js, Javascript, and HTML/CSS.

Utilized:
  • React.js
  • Javascript
  • HTML/CSS
  • APIs, Async, Authentication Flow

Steam Scraper

My Steam Scraper scrapes a popular game (Rust) server's website for recently banned players along with their Steam IDs. Using Steam's APIs, a GET request is sent with these players' IDs to get the information seen in the table. The back end server is made with Node.js, using Express, Request, and Axios. The front end is made with HTML/CSS and Javascript (jQuery, Bootstrap).

Utilized:
  • Javascript (Express, Request, Axios, jQuery)
  • HTML/CSS
  • APIs

ROS Projects

These two projects were created using ROS (Robot Operating System).

First, I created an autonomous differential drive robot. The robot has inertia, joints, and a lidar laser sensor. A map of the world was created using a technique called Gmapping, which uses laser range data to make a map. I added a global and local planner to the robot, which plan paths to travel to the set goal. Another package called AMCL was implemented to account for the drift of the robot due to minor inaccuracies in sensor data.

Secondly, I created a wall-following robot that follows the curve of a wall or track using a PID controller.

Utilized:
  • ROS
  • Linux
  • C++, Python
  • PID Control

Astro Aero Robotics Club (AARC)

AARC is a club that focuses on mobility systems, multi-agent systems, and systems that can survive in extreme environments. In the fall of 2020, we entered the 2021 NASA RASC-AL Special Edition competition. The task was to design a water extraction system for mars that must also identify the layers undeath the surface. We designed and built our machine in under six months and competed in Hampton, VA, against ten other universities. I was the team lead of the Control & Telemetry subteam, writing software for an Arduino to control stepper motors, sensors, a drill, heater, and more. I also created a GUI in MATLAB to communicate with the Arduino. Lastly, a Python script was made to display all system telemetry data.

Utilized:
  • C++, MATLAB, Python
  • Sensors
  • Mechanical Design
  • Teamwork
  • Leadership

Quadcopter Simulation

I created a quadcopter simulation in the game engine Unity. An xbox controller is used for controlling the quadcopter, which works by applying forces and torques to simulate real life. Control Of Inverted Pendulum

I created an inverted pendulum using a stepper motor, an encoder, a couple 3-D printed parts, and an EPS32 microcontroller. The encoder outputs the pendulum angle which is sent to a PID controller. The controller outputs an angular acceleration to the stepper motor, which applys a force to the pendulum, balancing it.

Utilized:
  • Unity
  • C#
  • PID Control
  • Physics, Dynamics
